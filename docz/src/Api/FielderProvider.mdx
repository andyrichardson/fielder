# FielderProvider

_Exposes a form instance [(FormState)](/api-reference/useForm#formstate) via React context._

```tsx
import { FielderProvider } from "fielder"
```

## Example Usage

```tsx
const formState = useForm()

return <FielderProvider value={formState}>{children}</FielderProvider>
```

## Props

### value **(required)**

_The [(FormState)](/api-reference/useForm#formstate) to be shared._

Type: `FormState`
