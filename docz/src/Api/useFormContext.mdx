# useFormContext

_Retrieves form state exposed via context._

```tsx
import { useFormContext } from "fielder"
```

## Example Usage

```tsx
const formState = useFormContext()

return <button disabled={!formState.isValid}>Submit</button>
```

## Response

_The state of the form [(FormState)](/api-reference/useForm#formstate) along with accessors and mutators._

Type; `FormState`

> See [useForm documentation](/api-reference/useForm) for more information about _FormState_

## Arguments

`useForm` doesn't currently take any arguments.
