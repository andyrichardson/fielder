# useForm

_Creates a new form instance._

```tsx
import { useForm } from 'fielder'; 
```

## Example Usage

```tsx
const formState = useForm();

return (
  <FielderProvider value={formState}>
    {children}
  </FielderProvider>  
)
```


## Response

_A `FormState` object (state of the form)._

`FormState`

## Arguments

`useForm` doesn't currently take any arguments.
